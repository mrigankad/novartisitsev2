# Novartis ITSE Dashboard v2

Operational Excellence Dashboard for Novartis ITSM data.

## Features
- Overview Dashboard with key KPIs (Ticket Volume, SLA Compliance, MTTR)
- Detailed "My Dashboard" with trend analysis
- Excel-style Leaderboards for resolvers and assignees
- Drill-down capabilities for ticket-level details
- Export to Excel and PDF functionality

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS for styling
- Radix UI for accessible components
- Recharts for data visualization
- Lucide React for iconography

## Getting Started
1. `npm install`
2. `npm run dev`
